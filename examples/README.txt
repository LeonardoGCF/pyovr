To run the examples:

* Install python 2.7
* Install Oculus runtime
* Install pyovr
* For the glut example, install GLUT and PyOpenGL
* For the pygame example, install pygame and cgkit
* For the pyside_qt example, install PySide
* For the glfw example, install glfw and pyglfw
* For the OpenSceneGraph example, install osgpypp

Then:

> python report_hmd_orientation.py
> python glut/rift_demo_glut.py
> python pygame/RiftDemo.py
> python pyside_qt/rift_demo_qt.py
> python glfw/rift_demo_glfw.py
[osg example is a work in progress]
